本程序使用VB编写，不使用任何驱动。所用到的API大多数为NTDLL的未被公开的函数，速度和功能比一般的任务管理器强大。
基本功能：
・多种枚举方法来枚举窗口，窗口无法隐藏
・多种枚举方法来枚举进程，可以找出隐藏进程
・窗口管理：激活/不激活；最大化/最小化；置顶/不置顶；查找窗口；多种方法关闭窗口，保证关闭任意窗口；查看窗口对应的进程和线程；查看窗口的状态和句柄；……
・进程管理：多种方式结束进程；查看进程各项属性（PID、优先级、命令行、EPROCESS）；查看进程对应的线程和窗口；进程挂起或恢复；简易的进程调试；……
・服务管理：与大多数服务管理器一样，可以查看各个服务
・注册表管理：与regedit一样
・系统杂项管理：SSDT表查看与恢复、模块查看和内存备份
